AbuFramework is modern & advanced Options Framework for WordPress themes and plugins.


== Description ==

AbuFramework is modern & advanced Options Framework for WordPress themes and plugins.
Created in OOP ( Object Oriented Programming ). 
It allows to create custom options, mataboxes, widgets, userprofile fields, widgets and taxonomies.


Features
1. Admin Option
2. MetaBoxes
3. TabOn
4. Taxonomy
5. userProfile
6. Widget


= Additional Features =
* Ajax Based Fetch html.
* Load needed Section only.
* Field Validation.
* Full value escaping.

== Installation ==

You just need to install AbuFramework. Options may created by your Plugin or Theme. 
 
e.g.
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
